Used within a control wrapper, these filters work with the controls to alter the chart's display.

All of the filters return a filter object, which is used with a ControlWrapper()

Category

{% highlight php startinline %} $categoryFilter = $lava->CategoryFilter('columnLabel'); // Lava::CategoryFilter() if using Laravel {% endhighlight %}

ChartRange

{% highlight php startinline %} $chartRange = $lava->ChartRangeFilter('columnLabel'); // Lava::ChartRangeFilter() if using Laravel {% endhighlight %}

DateRange

{% highlight php startinline %} $dateRange = $lava->DateRangeFilter('columnLabel'); // Lava::DateRangeFilter() if using Laravel {% endhighlight %}

NumberRange

{% highlight php startinline %} $numberRange = $lava->NumberRangeFilter('columnLabel'); // Lava::NumberRangeFilter() if using Laravel {% endhighlight %}

String

{% highlight php startinline %} $string = $lava->StringFilter('columnLabel'); // Lava::StringFilter() if using Laravel {% endhighlight %}
