Frequently Asked Questions

Here are some solutions to real world problems.

Formatting a single cell

If you don't want to apply a format to an entire column, you can manually create a DataCell object with the value and format string.

Example

{% highlight php startinline %} $data = $lava->DataTable(); $data->addNumberColumn('Values') ->addRow([15]) ->addRow([22]) ->addRow([26]) ->addRow($lava->DataCell(31, 'Thirty One')) ->addRow([43]) {% endhighlight %}

Outputting charts as static PNGs

This freature was commonly requested, and the javascript solution was a little convoluted, so now, it is a simple toggle.

It is as simple as adding 'png' => true to the options of the chart.

Example

{% highlight php startinline %} $data = $lava->DataTable(); /** Adding columns and rows */ $lava->LineChart('StaticChart', $data, [ 'png' => true ]); {% endhighlight %}

Outputting charts using Material styling

It is as simple as adding 'material' => true to the options of the chart.

Example

{% highlight php startinline %} $data = $lava->DataTable(); /** Adding columns and rows */ $lava->LineChart('StaticChart', $data, [ 'material' => true ]); {% endhighlight %}
Note: Only Line, Bar and Scatter charts are currently supported for Material design. Until Google updates their charts, I cannot force Material output.
