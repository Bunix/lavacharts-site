ComboChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $finances = $lava->DataTable(); $finances->addDateColumn('Year') ->addNumberColumn('Sales') ->addNumberColumn('Expenses') ->addNumberColumn('Net Worth') ->addRow(array('2009-1-1', 1100, 490, 1324)) ->addRow(array('2010-1-1', 1000, 400, 1524)) ->addRow(array('2011-1-1', 1400, 450, 1351)) ->addRow(array('2012-1-1', 1250, 600, 1243)) ->addRow(array('2013-1-1', 1100, 550, 1462)); $combochart = $lava->ComboChart('Finances') ->setOptions(array( 'datatable' => $finances, 'title' => 'Company Performance', 'titleTextStyle' => $lava->TextStyle(array( 'color' => 'rgb(123, 65, 89)', 'fontSize' => 16 )), 'legend' => $lava->Legend(array( 'position' => 'in' )), 'seriesType' => 'bars', 'series' => array( 2 => $lava->Series(array( 'type' => 'line' )) ) )); // Note: In the series config, '2' correlates to the 3rd horizontal axis dataset. // The DateColumn is the VerticalAxis, so 'Net Worth' is the 3rd, key 2, in the series. {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $combochart or $lava to your view to render.

View

Vanilla

{% highlight php startinline %}
// With the chart object render('finances-div') ?> // With the lava object render('ComboChart', 'Finances', 'finances-div') ?> {% endhighlight %}

Laravel

// With Lava class alias // With Blade Templates @combochart('Finances', 'finances-div') {% endhighlight %}
