ColumnChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $finances = $lava->DataTable(); $finances->addDateColumn('Year') ->addNumberColumn('Sales') ->addNumberColumn('Expenses') ->setDateTimeFormat('Y') ->addRow(array('2004', 1000, 400)) ->addRow(array('2005', 1170, 460)) ->addRow(array('2006', 660, 1120)) ->addRow(array('2007', 1030, 54)); $columnchart = $lava->ColumnChart('Finances') ->setOptions(array( 'datatable' => $finances, 'title' => 'Company Performance', 'titleTextStyle' => $lava->TextStyle(array( 'color' => '#eb6b2c', 'fontSize' => 14 )) )); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $columnchart or $lava to your view to render.
