PieChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $reasons = $lava->DataTable(); $reasons->addStringColumn('Reasons') ->addNumberColumn('Percent') ->addRow(['Check Reviews', 5]) ->addRow(['Watch Trailers', 2]) ->addRow(['See Actors Other Work', 4]) ->addRow(['Settle Argument', 89]); $lava->PieChart('IMDB', $reasons, [ 'title' => 'Reasons I visit IMDB', 'is3D' => true, 'slices' => [ ['offset' => 0.2], ['offset' => 0.25], ['offset' => 0.3] ] ]); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $piechart or $lava to your view to render.

View

Vanilla

{% highlight php startinline %}
// With the lava object render('PieChart', 'IMDB', 'chart-div') ?> {% endhighlight %}

Laravel

{% highlight php startinline %}
// With Lava class alias // With Blade Templates @piechart('IMDB', 'chart-div') {% endhighlight %}
