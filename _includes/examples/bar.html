BarChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $votes = $lava->DataTable(); $votes->addStringColumn('Food Poll') ->addNumberColumn('Votes') ->addRow(array('Tacos', rand(1000,5000))) ->addRow(array('Salad', rand(1000,5000))) ->addRow(array('Pizza', rand(1000,5000))) ->addRow(array('Apples', rand(1000,5000))) ->addRow(array('Fish', rand(1000,5000))); $lava->BarChart('Votes') ->setOptions(array( 'datatable' => $votes )); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $barchart or $lava to your view to render.
