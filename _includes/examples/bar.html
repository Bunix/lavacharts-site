BarChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $votes = $lava->DataTable(); $votes->addStringColumn('Food Poll') ->addNumberColumn('Votes') ->addRow(['Tacos', rand(1000,5000)]) ->addRow(['Salad', rand(1000,5000)]) ->addRow(['Pizza', rand(1000,5000)]) ->addRow(['Apples', rand(1000,5000)]) ->addRow(['Fish', rand(1000,5000)]); $lava->BarChart('Votes', $votes); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $barchart or $lava to your view to render.

View

Vanilla

{% highlight php startinline %}
// With the lava object render('BarChart', 'Food Poll', 'poll_div') ?> {% endhighlight %}

Laravel

// With Lava class alias // With Blade Templates @barchart('Food Poll', 'poll_div') {% endhighlight %}
