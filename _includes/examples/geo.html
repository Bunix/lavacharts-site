GeoChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $popularity = $lava->DataTable(); $popularity->addStringColumn('Country') ->addNumberColumn('Popularity') ->addRow(array('Germany', 200)) ->addRow(array('United States', 300)) ->addRow(array('Brazil', 400)) ->addRow(array('Canada', 500)) ->addRow(array('France', 600)) ->addRow(array('RU', 700)); $geochart = $lava->GeoChart('Popularity') ->setOptions(array( 'datatable' => $popularity )); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $geochart or $lava to your view to render.

View

Vanilla

{% highlight php startinline %}
// With the chart object render('pop-div') ?> // With the lava object render('GeoChart', 'Popularity', 'pop-div') ?> {% endhighlight %}
