Charts

Visualize your data

Create a new chart by calling it's name from the Lavacharts instance.

Lavacharts currently supports:

  • AreaChart
  • BarChart
  • CalendarChart
  • ColumnChart
  • ComboChart
  • DonutChart
  • GaugeChart
  • GeoChart
  • LineChart
  • PieChart
  • ScatterChart
  • TableChart
  • TreeMap

Example

{% highlight php startinline %} $lava->LineChart('UniqueTitle', $data [, $options [, 'element-id' ] ] ) {% endhighlight %}
