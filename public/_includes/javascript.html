Event Callbacks

Your callback is passed the javascript event object and a reference to the chart in which the event took place.

{% highlight javascript startinline %} function selectCallback (event, chart) { // Useful for using chart methods such as chart.getSelection(); } {% endhighlight %}

Helper Function

Also available for use in javascript, is the lava object which has helper functions.

The get function takes a chart label (the label you used when creating the chart) and returns the chart object.

{% highlight javascript startinline %} lava.get('myFancyChart') returns Chart {% endhighlight %}

This will let you access the chart outside of a chart event.
