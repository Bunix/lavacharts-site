All of the methods return the Axis object to enable the chaining of configuration options

Alternatively, you can build an associative array, using the method names as keys, and pass the array into the Axis's constructor

baselineColor(string $blc) returns Axis
direction(int $d) returns Axis
format(string $f) returns Axis
gridlines(array $g) returns Axis
minorGridlines(array $mg) returns Axis
logScale(bool $ls) returns Axis
textPosition(string $tp) returns Axis
title(string $t) returns Axis
titleTextStyle(TextStyle $tts) returns Axis
maxValue(int $mv) returns Axis
minValue(int $mv) returns Axis
viewWindowMode(string $vwm) returns Axis
viewWindow(array $vw) returns Axis
