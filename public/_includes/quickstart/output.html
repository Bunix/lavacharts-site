Rendering the chart

There are two ways to get your chart to display on your page.

  1. The library will generate only the javascript, and render into a div with a set elementId
  2. The library will generate everything you need, div and javascript.

Syntax: render(chartType, chartLabel, elementId [, array(height, width)])

{% highlight php startinline %} // Example #1, output into a div you already created //
echo Lava::render('LineChart', 'Stocks', 'myStocks'); // Example #2, have the library create the div echo Lava::render('LineChart', 'Stocks', 'stocks-div', array('height'=>300, 'width'=>600)); {% endhighlight %}
