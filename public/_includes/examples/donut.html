DonutChart Example

Controller

{% highlight php startinline %} use Khill\Lavacharts\Lavacharts; $lava = new Lavacharts; // See note below for Laravel $reasons = $lava->DataTable(); $reasons->addStringColumn('Reasons') ->addNumColumn('Percent') ->addRow(array('Check Reviews', 5)) ->addRow(array('Watch Trailers', 2)) ->addRow(array('See Actors Other Work', 4)) ->addRow(array('Settle Argument', 89)); $donutchart = $lava->DonutChart('IMDB') ->setOptions(array( 'datatable' => $reasons, 'title' => 'Reasons I visit IMDB' )); {% endhighlight %}

For Laravel, omit "$lava = new Lavacharts" and replace "$lava->" with "Lava::"

Otherwise, pass $donutchart or $lava to your view to render.

View

Vanilla

{% highlight php startinline %}
// With the chart object render('IMDB', 'chart-div') ?> // With the lava object render('DonutChart', 'IMDB', 'chart-div') ?> {% endhighlight %}

Laravel

// With Lava class alias // With Blade Templates @donutchart('IMDB', 'chart-div') {% endhighlight %}
