All the methods that return Chart enable the chaining of configuration options

Any of the method names can also be used as the key in an associative array, to pass into the chart's setOptions() method

areaOpacity(float $ao) returns Chart
axisTitlesPosition(string $atp) returns Chart
hAxis(HorizontalAxis $ha) returns Chart
interpolateNulls(bool $in) returns Chart
lineWidth(int $lw) returns Chart
pointSize(int $ps) returns Chart
vAxis(VerticalAxis $ha) returns Chart
